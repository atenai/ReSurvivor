# ReSurvivor

「Re：サバイバー」

（概要・テーマ）
この作品は、ゲームクリエイター甲子園に応募する為に、
デジタルハリウッド大学院のゲームメカニクスラボという授業で開発した作品です。
HAL時代に作成したサバイバーを当時は技術的に難しかったシステムを
実装したリメイク作品として開発するというテーマで制作しました。

(開発時期・期間)
デジタルハリウッド大学院1年生時の2020年6月～2020年12月(7ヶ月)

（開発ツール）
・ゲームエンジン：Unity(バージョン2019.1.4f1)
・言語：C# 
・開発環境：Visual Studio2017 
・3D データ制作：Blender ・一部はアセットストアを使用しています。
・2D データの一部はフリー素材を使用しています。
・BGM はフリー素材を使用しています。 
・SE はフリー素材を使用しています。 
・使用アセット :・LowPoly Environment Pack
		・Sci Fi Warrior PBR HP Polyart
		・Rockets, Missiles & Bombs - Cartoon Low Poly Pack
		・High Quality War Plane
		・Grenade pack free
		・Standard Assets(for Unity 2018.4)
		・Super Spitfire
		・Dustyroom Casual Game Sound - One Shot SFX Pack

（チームメンバー）
開発チーム（計5人）
・プランナー1人（担当）
・プログラマー2人
・デザイナー2人

（担当内容）
自分はディレクターとプログラマーとプランナーの3つを兼任しました。
プログラムはプレイヤーキャラクターと細かなシステム周りを開発しました。
プランニングは企画書の作成をしました。
マネジメントはチームの進捗状況の管理をやりました。

（ゲームシステム）
脱出地点を目指す横スクロールシューティングゲームです。
・脱出用飛行機に乗るとゲームクリアーになります。
・敵の攻撃を受けると体力が減ります。
・体力が0になるとゲームオーバーになります。
・地雷を踏むとダメージをくらいます。
・救急箱を取ると体力が回復します。
・敵の頭上にある白いゲージは敵の体力ゲージです。

(操作方法)
[キーボード]
スペースキー：ジャンプ
AとDキー：移動
Gキー：弾の発射
R：リロード
ESC：ゲーム終了

(起動方法)
「ReSurvivor(ビルド)」というフォルダーを開いてください。
その中にある、「ReSurvivor.exe」というアプリケーションをダブルクリックしてください。
